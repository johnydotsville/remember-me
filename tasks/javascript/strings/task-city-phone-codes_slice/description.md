У вас есть российские телефонные номера в формате `8 (XXX) YYY-YY-YY`, где:

* `8` — код страны,

* `(XXX)` — код города (3 цифры),

* `YYY-YY-YY` — локальный номер.

### Задача

- Напишите функцию **`getCityCode(phone)`**, которая возвращает код города (цифры внутри скобок).

- Используя эту функцию, выберите московские номера (коды - 495 и 499) в массив.
